<!-- SOBRE O PROJETO -->

<h2>About</h2>

<p>
  <strong>EN:</strong> Ghost Films is a horror movie catalog that features an interactive ghost on the login screen, creating a more immersive experience. After logging in, you can explore a selection of films, interact with each title through likes and dislikes, and watch the original trailers via the "watch" button. The project also allows you to switch between English and Portuguese, improving accessibility.
</p>

<p>
  <strong>PT:</strong> Ghost Films é um catálogo de filmes de terror que conta com um fantasma interativo na tela de login, criando uma experiência mais imersiva. Após o login, é possível explorar uma seleção de filmes, interagir com cada título por meio de likes e dislikes, além de assistir aos trailers originais pelo botão "assistir". O projeto também permite alternar entre inglês e português, melhorando a acessibilidade.
</p>

<!-- FUNCIONALIDADES -->

<h2>Features</h2>

<p><strong>EN:</strong></p>
  <ul>
    <li>Interactive login screen with an animated ghost;</li>
    <li>Contextual animations (the ghost looks down at the email field and closes its eyes over the password field);</li>
    <li>Horror movie catalog with hover animations;</li>
    <li>Like and dislike system for each movie;</li>
    <li>Watch button that opens external videos;</li>
    <li>Bilingual interface (English / Portuguese) with dynamic switching for each title;</li>
    <li>Responsive layout optimized for mobile and desktop devices.</li>
  </ul>

<p><strong>PT:</strong></p>
  <ul>
    <li>Tela de login interativa com um fantasma animado;</li>
    <li>Animações contextuais (olhar para baixo no e-mail e fechar os olhos na senha);</li>
    <li>Catálogo de filmes de terror com animações de hover;</li>
    <li>Sistema de likes e dislikes;</li>
    <li>Botão de assistir que abre vídeos externos;</li>
    <li>Interface bilíngue (inglês / português) com troca dinâmica para cada título;</li>
    <li>Layout responsivo otimizado para mobile e desktop.</li>
  </ul>

<!-- INTERAÇÕES -->

<h2>Interactions</h2>

<p><strong>EN:</strong></p>
  <ul>
    <li>When focusing the email field, the ghost looks down toward the input;</li>
    <li>When focusing the password field, the ghost closes its eyes to ensure privacy during password entry;</li>
    <li>Interact with the movies by liking or disliking them to register your preferences;</li>
    <li>Click the "Watch" button to open external videos in a new tab;</li>
    <li>Experience smooth animations when hovering over or clicking movie cards.</li>
  </ul>

<p><strong>PT:</strong></p>
  <ul>
    <li>Ao focar no campo de e-mail, o fantasma olha para baixo em direção ao campo;</li>
    <li>Ao focar no campo de senha, o fantasma fecha os olhos para garantir privacidade durante a digitação da senha;</li>
    <li>Interaja com os filmes dando like ou dislike para registrar suas preferências;</li>
    <li>Clique no botão "Assistir" para abrir vídeos externos em uma nova aba;</li>
    <li>Experimente animações suaves ao passar o mouse ou clicar sobre os cartões de filmes.</li>
  </ul>

<!-- PROPÓSITO -->

<h2>Purpose</h2>

<p>
<strong>EN:</strong> This project was developed with a focus on creating a front-end experience by designing a login screen that differs from traditional ones.
  The goal is to explore UX based on animations and interactions, making the experience more immersive on both desktop and mobile devices. During development,
  the objective was also to deepen the use of DOM manipulation and event-based interactions such as focus, hover, and click.
</p>

<p> 
<strong>PT:</strong> Este projeto foi desenvolvido com foco na criação de uma experiência de front-end, buscando criar uma tela de login diferente das tradicionais.
  A proposta é trabalhar UX baseada em animações e interações, tornando a experiência mais imersiva em dispositivos desktop e mobile. Durante o desenvolvimento,
  o objetivo também foi aprofundar o uso de manipulação do DOM e interações baseadas em eventos como foco, hover e clique.
</p>
